# Magical-Research
This project is a demo of the game Magic Research, developed entirely in Java for educational purposes, with the goal of studying and recreating the mechanics and structural design of the game’s early hours.

Legal notice: this is an amateur project created for educational purposes. All rights related to the original concept, gameplay, and idea belong to the developers of Magic Research.



Questo progetto è una demo del gioco Magic Research, sviluppata interamente in Java, con finalità didattiche, con l’obiettivo di studiare e ricreare le meccaniche e la struttura delle prime ore di gioco dell’opera originale.
